# notion_sync_data
将网络数据同步到notion笔记，方便快速查阅。

- 目前已支持获取豆瓣【影、音、书】的信息的获取
- 测试平台：Windows、Linux[Ubuntu20.4/Debain10]
- 开发环境：Python 3.10.2（建议使用使用版本Python3.8+）

> 该库是学习python做的自用小工具，代码参考于网络。（首个用python编写的工具，写的比较烂）


## 功能

- [x] 豆瓣【影、音、书】的信息
- [x] 创建notion数据库
- [x] 插入notion库
- [x] 已添加的不再重复添加（根据豆瓣链接判断）
